# California Housing (Spår A: Regression)

## Beskrivning
Denna notebook implementerar en komplett pipeline för att förutsäga bostadsvärden i Kalifornien baserat på områdesegenskaper

## Systemkrav
- Python 3.11
- Virtual environment (.venv)

## Installation
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Köra notebook
```bash
jupyter notebook ML_Uppgift1_Azar_Mohseni.ipynb
```

Klicka sedan på "Kernel" → "Restart & Run All" för att köra allt från början.
